# opencv-project
Face Detection and Face Recognition 

### Installation 
```
pip install opencv-contrib-python
```
[opencv-contrib-python documentation](https://pypi.org/project/opencv-contrib-python/)

#### Cascades used
1. frontal-face
2. eyes
3. nose
4. smile
